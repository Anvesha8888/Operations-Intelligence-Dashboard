# 📊 Operations Intelligence Dashboard (Excel)

## 📌 Project Overview

This project is an interactive **Operations Intelligence Dashboard** built in Microsoft Excel using the Sample Superstore dataset. It provides executive-level insights into sales performance, profitability, shipping efficiency, regional performance, and product analysis.

The dashboard is designed to help business stakeholders monitor key operational metrics and make data-driven decisions.

---

## 🎯 Business Objective

The objective of this dashboard is to:

- Monitor business performance
- Track key operational KPIs
- Analyze regional sales trends
- Evaluate product profitability
- Improve shipping efficiency
- Support executive decision-making

---

## 📂 Dataset Information

- **Dataset:** Sample Superstore
- **Rows:** 9,995
- **Columns:** 21
